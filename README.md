# Hosur All Property

Premium real estate website built for Hosur All Property.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM

## Features

- Premium luxury UI
- Responsive design
- Animated transitions
- Property showcase
- Legal pages
- WhatsApp integration
- Modern glassmorphism aesthetics

## Deployment

Hosted on Vercel.
